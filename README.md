1) Установить node.js
2) Установить gulp через терминал, например командой sudo npm i gulp -g
3) В каталоге проекта запустить команду npm i
4) Запускаем команду gulp в том же каталоге
5) В браузере сайт доступен через localhost:9451 либо можно открыть в браузере index.html из папки dev
6) Собранный и ужатый проект доступен в папке prod. Чтобы внести обновления в нее, необходимо запустить команду gulp prod через терминал
